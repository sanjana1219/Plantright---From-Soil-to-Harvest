# Plantright-From Soil to Harvest
Crop Recommendation System using Data Mining

Project Overview
Plantright is a data-driven crop recommendation system developed to support smarter agricultural decision-making. The project uses data mining techniques to predict the most suitable crops based on soil composition and environmental conditions such as nutrient levels, temperature, humidity, pH, and rainfall.

The system applies the K-Nearest Neighbors (KNN) classification algorithm to analyze agricultural data and generate accurate crop recommendations. By combining statistical analysis and machine learning, this project helps farmers improve productivity, optimize resources, and promote sustainable farming practices.
